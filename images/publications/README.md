# Publication thumbnails

Put each paper's preview image in this folder, then uncomment the matching
`image:` line in `_data/publications.yml`.

Expected files (rename your images to match, or edit the paths in the YAML):

| Paper            | Filename            |
|------------------|---------------------|
| ChinaHeritaQA    | `chinaheritaqa.png` |
| LLM4Brain (AAAI) | `llm4brain.png`     |
| M-ABSA (EMNLP)   | `mabsa.png`         |
| MTFNet (APSIPA)  | `mtfnet.png`        |

Recommended size: roughly 360×220 px (a 16:10-ish landscape crop); they are
displayed at 180×110 px on the homepage. PNG or JPG both work.
